A high performance, open-source universal RPC framework

Why HRPC?

HRPC is a simple open source RPC framework that can run in any environment. It can efficiently connect services in and across data centers with pluggable support for authentication. It is also applicable in last mile of distributed computing to connect devices, mobile applications and browsers to backend services.
Simple service definition
Define your service using Protocol Buffers, a powerful binary serialization toolset and language
Start quickly and scale
Install runtime and dev environments with a single line and also scale to millions of RPCs per second with the framework
Works across languages and platforms
Automatically generate idiomatic client and server stubs for your service in a variety of languages and platforms
Integrated auth
Fully integrated pluggable authentication with http/1 based transport
